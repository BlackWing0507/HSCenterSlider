## Installation

HSCenterSlider is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'HSCenterSlider'
```
##### Possible Error:
`[!] Unable to find a specification for 'HSRange'` 
##### Solution:

```ruby
pod setup
```
